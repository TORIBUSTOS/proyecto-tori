TORO INVESTMENT MANAGER
======================

INICIO DEL SISTEMA
-----------------
Usar SOLO uno de estos archivos según el caso:

1) Desarrollo / cambios de código
   INICIAR_TORO_DEV.bat

2) Uso normal / sistema estable
   INICIAR_TORO_PROD.bat


ACCESO AL SISTEMA
-----------------
Abrir el navegador y entrar a:
http://localhost:8000


SECCIONES DISPONIBLES
--------------------
- Inicio:     http://localhost:8000
- Reportes:   http://localhost:8000/reportes
- Batches:    http://localhost:8000/batches
- API Docs:   http://localhost:8000/docs


CÓMO CERRAR EL SISTEMA
---------------------
En la ventana negra del sistema:
CTRL + C


NOTAS IMPORTANTES
-----------------
- No ejecutar comandos manualmente si se usan los .bat
- No usar el modo DEV para dejar el sistema corriendo todo el día
- Si el sistema no abre, revisar la ventana negra por errores
